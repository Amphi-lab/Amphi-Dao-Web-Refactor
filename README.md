# Build project
```shell
pnpm install
```


# Run project
```shell
npm run dev
```


# [Storybook](https://juejin.cn/post/7044428114264326175)
## 安装storybook/cli脚手架并安装依赖
```shell
pnpm dlx storybook@latest init
```

## 启动你的storybook组件库项目
```shell
yarn storybook
```
or use 
```shell
npm run storybook 
```
