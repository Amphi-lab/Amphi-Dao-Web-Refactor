# [Storybook](https://juejin.cn/post/7044428114264326175)
## 安装storybook/cli脚手架并安装依赖
```shell
npx -p @storybook/cli sb init --type react_scripts
```

## 启动你的storybook组件库项目
```shell
yarn storybook 1
```
or use 
```shell
npm run storybook 
```
