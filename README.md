# Build project
```shell
pnpm install
```


# Run project
```shell
npm run dev
```

# [Storybook](https://storybook.js.org/docs/react/get-started/install)
```shell
pnpm dlx storybook@latest init
```

## 启动你的storybook组件库项目
```shell
yarn storybook
```

# Test Tool: [Cypress](https://www.cypress.io)
```shell
npm install cypress --save-dev
```

```shell (after start npm run dev)
npx cypress open
```
